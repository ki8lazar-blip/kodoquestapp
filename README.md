# 🚀 KodoQuest

Εκπαιδευτικό παιχνίδι προγραμματισμού για παιδιά Δημοτικού.

## Χαρακτηριστικά
- Επιλογή γλώσσας (Ελληνικά / Αγγλικά)
- Επιλογή τάξης (Α΄–Στ΄ Δημοτικού) με προσαρμοσμένο επίπεδο δυσκολίας
- 4 χαρακτήρες προς επιλογή
- 10 αποστολές: Drag & Drop → Ακολουθίες → Βρόχοι → Συνθήκες → Αλγόριθμοι
- Σύστημα ζωών και αστεριών
- Πλήρης υποστήριξη αφής (tablet/κινητό)

## Τεχνολογίες
- Καθαρό HTML / CSS / JavaScript (χωρίς εξαρτήσεις)
- Ένα μόνο αρχείο `index.html`
